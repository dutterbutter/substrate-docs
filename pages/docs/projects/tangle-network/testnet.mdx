# Incentivized Testnet Campaign

In order to ensure the security, privacy and robustness of the Tangle Network, we are planning an incentivized usage and testing campaign to onboard developers, enthusiasts and researchers to test our open-source technologies.

### Points System

Users can participate by engaging in the following activities, and if needed, reporting their contributions to [our Discord](https://webb.tools/community) in the `#contributions` channel.

| Task                                           | Points   | Type            | Instructions                                                                                                                                       |
| ---------------------------------------------- | -------- | --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| Creative Contribution                          | Variable | User Submission | -                                                                                                                                                  |
| Use Case Concepts                              | 10       | User Submission | Write an article discussing a use case for the technologies Tangle uses!                                                                           |
| Publish a video or tutorial on Tangle/Webb     | 20       | User Submission | Publish a high-quality video discussing the network or its privacy technologies.                                                                   |
| Author a Protocol Extension                    | 3        | User Submission | Create and submit an extension to the protocols used in Tangle. See our GitHub                                                                     |
| Tweet a Semaphore identity w/ Webb             | 20       | User Submission | Share your [Semaphore identity](https://semaphore.appliedzkp.org/docs/guides/identities) and tag [@webbprotocol](https://twitter.com/webbprotocol) |
| DKG Authority (get voted in)                   | 5        | Automatic       | See our DKG documentation                                                                                                                          |
| Tweet about Webb                               | 3        | User Submission | Tag [@webbprotocol](https://twitter.com/webbprotocol) in a high quality tweet about the network.                                                   |
| Deposit → Transfer → Withdraw                  | 10       | Automatic       | Perform a transaction flow on the network or the Hubble Bridge.                                                                                    |
| Tweet a hash of your address + a secret number | 20       | User Submission | Make sure to tag @Webbprotocol.                                                                                                                    |
| Vote in Governance                             | 1        | Automatic       | See the Polkadot Governance Docs for [how to participate in Substrate governance.](https://wiki.polkadot.network/docs/maintain-guides-democracy)   |
| Propose and Pass in Governance                 | 1        | Automatic       | See the Polkadot Governance Docs for [how to participate in Substrate governance.](https://wiki.polkadot.network/docs/maintain-guides-democracy)   |
| Tweet an ETH address or ENS domain             | 15       | User Submission | Tag [@webbprotocol](https://twitter.com/webbprotocol) in a high quality tweet about the network.                                                   |
| Bug Bounty with PR Fix Submission              | 50       | User Submission | See our Docs on Contributing a Bug Bounty, and provide a fix for the bug via Github.                                                               |
| Run a validator                                | 1        | Automatic       | See our Docs on running a Validator Node                                                                                                           |
| Scripts to make Transactions                   | 10       | User Submission | Engineer and publish transaction scripts for the network                                                                                           |
| Bug Report                                     | 10       | User Submission | See the @Bug Reporting Guidance                                                                                                                    |
